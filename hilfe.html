SelfPanini

Hilfe & Datenschutz – einfach erklärt

SelfPanini ist ein kostenloser Helfer, um dein WM-2026-Sammelalbum zu verwalten: Sticker abhaken, Fehlende und Doppelte im Blick behalten und mit Freunden tauschen. Kein Konto, keine Anmeldung, keine Werbung.

01So funktioniert's

  • Antippen = hab ich (grün). Nochmal antippen = doppelt (gelb). Lange drücken = einen zurück.
  • Teilen: Deine Listen „fehlt / doppelt" per WhatsApp verschicken oder einen Tausch-Link erstellen – der Freund sieht sofort, was ihr tauschen könnt.
  • Speichern: Alles wird automatisch gespeichert. Mit dem Speichern-Knopf kannst du zusätzlich eine Sicherungsdatei ablegen.

02Wo wird gespeichert?

Ohne Cloud-Sync (Standard): Alles bleibt nur auf deinem Gerät, im Speicher deines Browsers. Nichts verlässt dein Gerät. Ein anderes Gerät weiß davon nichts.

Mit Cloud-Sync (freiwillig, du schaltest es selbst ein): Damit dieselbe Sammlung auf Handy und PC erscheint, wird dein Album-Stand zusätzlich bei unserem Dienstleister Cloudflare gespeichert – abgelegt unter deinem Sync-Code (z. B. PANINI-7F3KQ2). Nur wer diesen Code hat, kommt an die Daten.

03Was gespeichert wird – und was nicht

Das wird gespeichert

  • Für jeden Sticker eine Zahl: 0 = fehlt, 1 = hab ich, 2+ = doppelt (z. B. GER 4 = 2)
  • Dein frei gewählter Spitzname, falls du einen eingibst
  • Dein Sync-Code (nur bei aktivierter Cloud-Sync)

Das wird NICHT gespeichert

  • Kein echter Name, keine Adresse, keine E-Mail, keine Telefonnummer
  • Kein Login, kein Passwort
  • Kein Standort, keine Kontakte, keine Fotos
  • Keine Werbe-Cookies, kein Tracking, keine Weitergabe oder Verkauf von Daten

Kurz: Es werden nur Zahlen zu Stickern gespeichert – nichts, woran man erkennt, wer du bist. Bitte trage als Spitznamen keinen echten vollständigen Namen ein.

Wichtig: Der Sync-Code ist kein Passwort. Wer den Code kennt, kann dein Album sehen und ändern. Teile ihn deshalb nur mit deinen eigenen Geräten oder mit Freunden, mit denen du bewusst gemeinsam sammelst.

04Löschen & Kontrolle

  • Auf dem Gerät löschen: „Mehr → Alles zurücksetzen".
  • Aus der Cloud löschen: „Mehr → Cloud-Sync → Sammlung aus Cloud löschen". Danach ist die Cloud-Kopie leer.
  • Cloud gar nicht nutzen: Einfach keinen Sync-Code erstellen – dann ist nie etwas online.

05Für Eltern

SelfPanini ist bewusst simpel und sicher gebaut: kein Konto, keine Registrierung, kein Chat mit Fremden, keine Werbung, keine In-App-Käufe. Kinder tauschen Codes nur mit Freunden, die sie kennen. Es werden keine persönlichen Daten abgefragt.
